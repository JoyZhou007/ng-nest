# Container 布局容器

用于布局的容器组件，方便快速搭建页面的基本结构：

- `<x-container>`：外层容器。当子元素中包含 `<x-header>` 或 `<x-footer>` 时，全部子元素会垂直上下排列，否则会水平左右排列。
- `<x-header>`：顶栏容器。
- `<x-aside>`：侧边栏容器。
- `<x-main>`：主要区域容器。
- `<x-footer>`：底栏容器。
